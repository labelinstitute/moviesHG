Supplementary Network Animations

Interactive simulations illustrating equilibrium and non-equilibrium dynamics in small networks. Each file contains a step-by-step visualization of opinion evolution over time.

Equilibrium Dynamics

  • Equilibrium Eq_5Ha.html
    Stable convergence to consensus in a 5H configuration. The system reaches a steady state after a finite number of updates.
  • Equilibrium Eq_3Hb.html
    Convergence to equilibrium in a 3H variant (type b). Demonstrates stable consensus under local interactions.
  • Equilibrium Eq _3Lb.html
    Equilibrium case in a 3L configuration (type b). Shows gradual stabilization toward a fixed opinion distribution.

Non-Equilibrium Dynamics

  • Non-Equilibrium noEq_3La.html
    Persistent fluctuations in a 3L configuration (type a). The system does not settle into a stable consensus.
  • Non-Equilibrium noEq_5Ha.html
    Oscillatory dynamics in a 5H configuration. Opinions continue changing without reaching equilibrium.
